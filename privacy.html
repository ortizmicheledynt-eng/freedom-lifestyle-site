Privacy Policy

Last updated: January 1, 2025  ·  Freedom Lifestyle

Summary: This tool is for internal use only by Freedom Lifestyle. We do not collect, store, or share any personal data. Your Pinterest API credentials are stored exclusively on your own device.

1. Who we are

Freedom Lifestyle operates this internal keyword research tool (the "Tool") to assist with Pinterest advertising campaign management. This Privacy Policy explains how we handle information in connection with the Tool.

This Tool is not a public-facing product and is intended solely for internal use by Freedom Lifestyle personnel.

2. What information the Tool accesses

The Tool connects to the Pinterest Ads API using credentials that you provide. Specifically, it accesses:

  • Pinterest keyword suggestion data for your own ad account
  • Monthly search volume estimates for keywords (as provided by Pinterest)

The Tool does not access, collect, or process any personally identifiable information about Pinterest users or any third parties.

3. How we handle your credentials

Your Pinterest Ads API access token and Ad Account ID are:

  • Stored exclusively on your local device using browser local storage
  • Never transmitted to any server operated by Freedom Lifestyle or any third party
  • Used only to make direct requests to the Pinterest API on your behalf
  • Removable at any time by clearing your browser extension data

We do not have access to your credentials. They exist only on your device.

4. Data storage and retention

The Tool does not operate any servers or databases. No data retrieved from the Pinterest API is stored beyond your active browser session. Keyword results are displayed in the Tool interface and discarded when the session ends or the extension is closed.

The only persistent data is your API credentials stored locally on your device, which you control entirely.

5. Third-party services

The Tool communicates exclusively with the Pinterest Ads API (api.pinterest.com). No other third-party services, analytics tools, or tracking scripts are used.

Your use of the Pinterest API is also governed by Pinterest's own Developer and API Terms of Service and Privacy Policy.

6. Pinterest API compliance

This Tool is built in compliance with Pinterest's Developer Guidelines. Specifically:

  • It accesses only the ad account belonging to Freedom Lifestyle
  • API credentials are stored locally (not server-side), per Pinterest's "bring-your-own-key" policy
  • No data retrieved from the Pinterest API is shared with or sold to third parties
  • The Tool is used solely for managing and researching our own Pinterest advertising campaigns
  • No data is used to target advertising outside of Pinterest

7. Cookies

This website and the Tool do not use cookies or any tracking technologies.

8. Children's privacy

This Tool is intended for internal business use only and is not directed at or accessible to children under the age of 13.

9. Changes to this policy

We may update this Privacy Policy from time to time. Any changes will be reflected on this page with an updated date. Continued use of the Tool after changes constitutes acceptance of the revised policy.
