Hướng dẫn deploy Cloudflare Worker
1. Vào workers.cloudflare.com → đăng ký miễn phí
2. Nhấn Create Worker → đặt tên (vd: pinterest-proxy)
3. Xoá hết code mặc định → paste toàn bộ code từ file pinterest-worker.js vào
4. Nhấn Deploy → copy URL worker (dạng https://pinterest-proxy.xxx.workers.dev)
5. Paste URL đó vào ô Worker URL bên trên → Lưu
